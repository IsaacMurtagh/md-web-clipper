# Markdown Web Clipper

A Chrome extension that lets you pick any element on a page and copy it to your clipboard as Markdown.

## How it works

1. Click the extension icon and hit **Pick Element**
2. Hover over the page — elements highlight as you move
3. Click an element to convert it to Markdown and copy to clipboard
4. Press **Escape** to cancel

## Install from source

1. Clone this repo
2. Open `chrome://extensions/` and enable **Developer mode**
3. Click **Load unpacked** and select the project folder
4. Navigate to any page and click the extension icon

## Third-party

This extension bundles [Turndown](https://github.com/mixmark-io/turndown) and [turndown-plugin-gfm](https://github.com/mixmark-io/turndown-plugin-gfm) by Dom Christie (MIT License) for HTML-to-Markdown conversion.
